CSS

body {
background-color: rgb(34,139,34);
font-family: "calibri";
font-size: 20px;
}

Inde på css dokumentet kan du bla. inden for <body> lave om på hjemmesidens baggrund, tekst-type, tekststørelse osv.

Som du kan se har jeg ændret baggrunden, ved hjælp af background-color: rgb(34,139,34);, til forest-green. Dette gøres inde i rgb, som betyder red green blue, hvilket er mængden af rød grøn og blå du lægger ind for t skabe en farve.

font-family: "calibri"; ændrer på fonten af den tekst du lægger ind i hjemmesiden. Du kan ændre på fonten af din hjemmeside der hvor der står calibri.

font-size: 20px; ændrer på tekstens størelse, hvilket du kan modifisere der hvor der står 20px. Slet ikke px'et, for så fungerer braset ikke.


div {
width: 600px;
height:500px;
border: 0px solid green;
padding: 0px;
margin: auto;
background-color: rgb(255, 255, 255);
}

div{} bestemmer hvor indeholdet af en hjemmeside skal være.

Som du kan se har jeg sat width og height til 600px og 500 px respektivt. Dette ændrer på bredden og højden af den boks hvor din tekst kan være.

Border: 0px solid green; ændrer på boksens omrids. Ting du kan ændre ved border er: tykkelsen og stilen(altså om du f.eks. vil have solid eller dotted border).

Padding: 0px; ændrer på distancen mellem tekst og border. I dette tilfælde er det 0.

margin: auto; ændrer på placeringen af boksen. I dette tilfælde er den i midten af hjemmesiden.
