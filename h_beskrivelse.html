HTML

<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
</head>
<body>
Hello world.
</body>
</html>

Til at starte med vil jeg beskrive et af de mest simple koder for en html hjemmeside der er.

<head> viser html dokumentets informationer. F.eks. at man kan bruge specialtegn, som æ,ø og å, inde i dokumantet( <meta charset="UTF-8">). Under <head> ser vi <body>, hvilket er hvor du opbygger din hjemmeside.

Hvis du gerne vil have en bestemt stil til hjemmesiden, kan du linke en css hjemmeside til din html hjemmeside. Dette kan du gøre ved at skrive <link rel="stylesheet" href="Navn på dit css dokument.css">.
